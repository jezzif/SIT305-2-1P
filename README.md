## About
This project is an android app that is a measurement convertor. It has been written in Java and is compatible with all versions of android from Android 7.0 onwards.

## Usage
The first drop down box allows switching between different units of measurement. Then you must pick the unit that you will be converting from, and input the value you want to convert. Then choose the unit that the value will be converted to and press the convert button.
